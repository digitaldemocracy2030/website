# 実証実験に参画されたい方

自治体・政党・企業団体の方は、デジタル民主主義2030プロジェクトの取り組みを導入して、市民参加を促進できます。

[Googleフォーム](https://docs.google.com/forms/d/e/1FAIpQLScytDD9cdYwcl_Gsr0qal-Op7mpXaQzK3KGTk_FsahY5FkU9w/viewform?usp=header)よりご連絡下さい。


# 寄付にご協力いただける方

寄付担当窓口の塔本 donation@dd2030.org までご連絡ください


# 開発・デザイン・改善・コミュニティ運営に興味のある方

<a href={slack_join_link}>Slackに参加</a>して様子を見てみましょう。
